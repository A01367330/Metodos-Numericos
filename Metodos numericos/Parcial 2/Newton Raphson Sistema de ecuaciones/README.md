#NEWTON RPHSON SE

Consiste en elegir las coordenadas de un punto (x1, y1) como aproximación del punto de intersección de las funciones u(x, y) y v(x,y) que hacen que estas se anulen

x= x1- (((udv/dy)-(vdu/dy))/j)          

y= y1- (((vdu/dx)-(udv/dx))/j)
